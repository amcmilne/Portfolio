Brewery Finder

|Group Project #1|

Brewery-Finder

The Brewery Finder application allows the user to search for information on breweries by city and state and filter the response by type (e.g. micro brewery, nano brewery, bar, etc.). The application will also automatically render results for the user's current location, if any are nearby. Additionally, if the user desires information for a specific brewery, they may make a search by name. The page will only display five results at most, but a user may paginate through the results using the next and previous buttons located at the bottom of the page. For each displayed brewery, the name, address, phone number, and website of the brewery will be provided if available from the database. A random brewery-related image from Pexels is appended to each rendered brewery section as well.

Weather Dashboard

|Homework Assignment|

Weather-Dashboard

Weather Dashboard allows a user to search for the weather by city. The search results will render the current and future conditions.The user is presented with the city name, the date, the temperature, the humidity, the wind speed, and the UV index. The user is also presented with the 5 days forecast that displays the date, the temperature, and the humidity. The users search will be stored, and when the search history is clicked, they will be presented with the weather for the city selected. When the weather dashboard is opened, the use is presented with the last searched city.

Day Planner

|Homework Assignment|

Day-Planner

The user is presented with a day planner. The day planner will be utilized to make a daily schedule. The user will have time blocks for each hour. The time blocks are colored coded to represent if the time is past, present, or future. The user can input their event into the timeblock and click save. The info will be saved to local storage, and will render upon refresh of the page.
